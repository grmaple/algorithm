### 2替换空格

##### 题目

用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。

##### 解法

```python
# -*- coding:utf-8 -*-
class Solution:
    def __init__(self):
        self.acceptStack = []
        self.outputStack = []
    def push(self, node):
        # write code here
        self.acceptStack.append(node)
    def pop(self):
        #outputStack为空就看看acceptStack能不能加进去
        if self.outputStack == []:
            while self.acceptStack:
                self.outputStack.append(self.acceptStack.pop())
        #outputStack要么空要么不空
        if self.outputStack:
            return self.outputStack.pop()
        else:
            return None   
```
